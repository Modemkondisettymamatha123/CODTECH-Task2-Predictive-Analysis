# Predictive Analysis using Machine Learning

## Internship Task - CODTECH (Task 2)

### Objective
To build a machine learning model for predictive analysis using a disease dataset.

### Dataset
- Diabetes Disease Dataset
- Target Variable: Outcome (Diabetic / Non-Diabetic)

### Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook

### Machine Learning Model
- Logistic Regression

### Steps Performed
- Data Loading and Exploration
- Feature Selection
- Train-Test Split
- Model Training
- Model Evaluation

### Results
The model successfully predicts disease outcomes with good accuracy.
